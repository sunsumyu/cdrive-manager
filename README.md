# C 盘空间管理器

一个用 Rust 编写的桌面 GUI 磁盘空间分析工具，参考了 WinDirStat、QDirStat、Filelight、ncdu、Diskonaut、duc 和 BleachBit 的设计思路。

当前版本是安全优先的 MVP：**只扫描和可视化，不删除任何文件**。

## 功能

- 默认扫描 `C:\`，也可以输入其他目录。
- 支持系统目录选择窗口选择扫描目录。
- 开始扫描前会校验路径是否存在、是否是目录。
- 后台线程扫描，界面不会因为扫描 C 盘而完全卡死。
- 支持取消正在进行的扫描，并保留已扫描到的部分结果。
- 统计：
  - 累计大小
  - 文件数量
  - 目录数量
  - 访问错误数量
- 显示：
  - 最大目录
  - 最大文件
  - 文件扩展名/类型占用
  - 访问错误列表
- Treemap 矩形空间占用图，参考 WinDirStat / QDirStat。
  - 支持点击目录块进入子目录视图。
  - 支持返回上级和返回根目录。
  - 支持在目录块右键复制路径或打开位置。
  - 当前 Treemap 下钻基于已保留的最大目录结果，可能不是完整目录树。
- 在最大目录/最大文件列表中支持：
  - 复制完整路径
  - 打开目录或文件所在位置
- 默认不跟随符号链接，降低循环扫描风险。
- 遇到无权限目录会记录错误并继续扫描。

## 运行

```bash
cargo run --release
```

首次扫描整个 C 盘可能耗时较长。建议先扫描一个较小目录测试，例如：

```text
C:\Users\你的用户名\Downloads
```

如果扫描目录选错，点击“取消扫描”即可停止后续扫描。取消后界面会显示已经统计到的部分结果。

## 安全策略

当前版本不会删除、移动或修改任何文件。

后续如果添加清理功能，应遵循：

1. 默认 dry-run 预览。
2. 删除前显示预计释放空间。
3. 默认移动到回收站，而不是永久删除。
4. 系统目录和用户文档目录默认保护。
5. 高风险清理规则必须二次确认。
